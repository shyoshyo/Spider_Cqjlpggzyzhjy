# spider for cqjlpggzyzhjy.gov.cn cqjlpggzyzhjy.gov.cn 爬虫

## Requirement 依赖
  - Python3 (3.6.4)
  - Scrapy (1.5.0)
  - BeautifulSoup4 (4.6.0)


## Usage 使用方法

  - Enter Project Directory 进入工程目录

    ```
    $ cd path/to/directory
    ```

  - Start Scrapy framework 启动 Scrapy 框架 

    ```
    $ scrapy crawl cqjlpggzyzhjy
    ```
